Automaton Simulator tools

Using the Automaton Simulator toolbar involves three tools. Click on a tool button to select it; the program indicates your current selection by giving the button a dark background.

The four buttons, going left to right in the toolbar, are the following:

  • The state tool, for adding states to the automaton.

  • The transition tool, for adding transitions to the automaton.

  • The text tool, for adding text labels to the automaton.

  • The simulation buttons, for simulating the automaton on a string. (Here, there is only one simulation button. For Turing machines, there will be five.)
